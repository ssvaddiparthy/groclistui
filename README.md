# GrocListUI

This react.js project serves as the home for a WEB UI for GrocList Rest API

#### Development Guide:
1. Clone the project and run:
    1. `npm install`
    1. `npm start`
1. Standard React practices
1. Run `prettier --write src/*.js` before a code push to maintain clean code
